What is it?
***********

**CapyrX** is a GPU-accelerated multipatch infrastructure, written in C++ for the `CarpetX <https://github.com/eschnett/CarpetX>`_ AMR driver, which is intended for the `Einstein Toolkit <https://einsteintoolkit.org/>`_. **CarpetX** is based on `AMReX <https://amrex-codes.github.io>`_, a software framework for block-structured AMR (adaptive mesh refinement).
